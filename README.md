# Service Operations Control Tower Dashboard

## Overview

Built an operations dashboard to monitor patient volume, shift demand, risk levels, repeat users, and throughput performance.

## Business Objective

Support management decisions for staffing allocation, workload balancing, and operational efficiency.

## Tools Used

* Microsoft Excel
* Power Query
* Pivot Tables
* KPI Cards
* Dashboard Design
* Slicers

## Key Questions Answered

* When is peak workload?
* Which shifts need more staffing?
* Which users create repeat demand?
* Which age groups drive demand?
* Where are high-risk cases concentrated?

## Key Insights

* Peak demand occurred between 11:00–18:00.
* Repeat users represented a major share of demand.
* Patients aged 51+ drove highest volume.
* Discharges exceeded admits in most periods.
* Certain shifts handled higher-risk cases.

## Business Recommendations

* Reallocate staff during peak windows.
* Build repeat-user intervention programs.
* Improve night-shift risk readiness.
* Maintain discharge efficiency.
